Import Emissions Data

Upload a CSV, XLSX, JSON, XML file containing your emissions data. Your file must include the required columns.

☁️
Click to upload or drag and drop
CSV, XLSX, JSON, XML files up to 100MB
Need the correct format?

Required Columns:

  • scope_type - Scope (1, 2, 3, etc.)
  • category - Category (natural_gas, etc.)
  • value - Emission value
  • unit - Unit (m3, kWh, kg, etc.)
  • reporting_period - YYYY-MM-DD
  • emission_date - YYYY-MM-DD

Optional Columns:

  • region_code - Region (US, EU, etc.)
  • source - Source description
  • location - Location name
  • subcategory - Detailed category
  • supplier_id - Supplier ID
  • material_code - Material code
  • notes - Additional notes
